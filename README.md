# AR Fantasy Scavenger Hunt

A cross-platform mobile AR scavenger hunt game built with Unity and AR Foundation.

## Features

- **Augmented Reality**: Uses AR Foundation for iOS (ARKit) and Android (ARCore)
- **Fantasy Theme**: Collect magical crystals, runes, and mystical items
- **Rarity System**: Common → Uncommon → Rare → Epic → Legendary items
- **Timed Challenges**: Hunt against the clock with time bonuses
- **Achievements**: Unlock achievements as you collect
- **Collection Journal**: Track your discovered items
- **Progression**: Complete hunts to unlock new challenges

## Setup

1. Open Unity 2022.3 LTS
2. Create a new 3D (URP) project
3. Copy `Assets/_Project/` into your project's Assets folder
4. Run editor automation: **Window → AR Fantasy → Setup Scene**
5. Run: **Window → AR Fantasy → Create Sample ItemData**
6. Run: **Window → AR Fantasy → Create Sample HuntConfigs**
7. Run: **Window → AR Fantasy → Create Crystal Prefab**

## Build

- **Android**: `Window → AR Fantasy → Build → Android (APK)`
- **iOS**: `Window → AR Fantasy → Build → iOS`

## Requirements

- Unity 2022.3 LTS
- AR Foundation package
- ARCore XR Plugin (Android) or ARKit XR Plugin (iOS)
- AR-compatible device (iOS 14+ or Android 7.0+)
